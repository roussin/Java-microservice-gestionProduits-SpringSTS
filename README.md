# Java-microservice-gestionProduits-SpringSTS

Un microservice pour la gestion de produits en Java :

- Tester l'API avec postman
- Filtrer les réponses
- JPA/H2 communiquer avec la base de donnée importé H2
- Gérer les erreurs et valider les données
- Swagg2 documenter le microservice, configuration personnalisée Swagg2
